codigo fonte / pre formatação

O comando <code> faz com que a fonte fique monoespaçada facilitando a leitura de codigos, porem ela nao faz a formatação de um codigo, como quebras de linha e tudo mais, exemplos:
o trecho a seguir esta dentro da tag code document.getElementByID('teste')
o comando a baixo tambem esta dentro da tag code, pore, por mais que no arquivo html esteja formatado, o simples fato de colocar na tag code, nao fara com que fique com a identação correta, exemplo:
num = int(input('digite um numero')) if num % 2 ==0: print(f'o numero {num} e PAR') else: print(f'o numero {num} é IMPAR)

Existe uma tag chamada <pre>, tudo o que estiver dentro dela sera considerado espaços tabulações e etc,entao a junção dela com a tag code, faz com que a exibição fique com uma aparencia melhor ja que o codigo estara identado da mesma forma que identou em seu arquivo HTML por exemplo:
